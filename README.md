# Guitar Practice

Практика-компаньон для гитары: метроном, тренажёр CAGED, тренажёр боксов ладов
с секвенциями и секундомер. PWA на чистых HTML/CSS/JS, без сборки и бэкенда,
работает офлайн. Ориентация — только альбомная (для iPad на подставке).

## Структура репозитория

```
index.html              — разметка всех экранов
css/styles.css          — стили (только landscape)
js/
  app.js                — навигация между экранами
  metronome.js           — метроном (Web Audio, look-ahead scheduling)
  stopwatch.js           — секундомер
  theory.js              — табличная транспонизация аккордов + квинтовый круг
  circle-of-fifths.js    — SVG-рендер квинтового круга
  fretboard.js           — общий SVG-рендерer грифа (CAGED и боксы)
  caged-screen.js        — экран CAGED (формы C-A-G-E-D + транспонизация)
  boxes-screen.js        — экран боксов + секвенций
  data/
    caged.js             — оцифрованные формы CAGED (по reference/major-form-*.png)
    boxes.js             — оцифрованные 7 боксов ладов (по reference/box-minor-major-*.png)
    sequences.js          — конфиг секвенций (наполняется вручную)
manifest.json, service-worker.js, icons/ — PWA
reference/               — исходные скриншоты аппликатур/боксов/круга квинт
tests/theory.test.js     — unit-тесты транспонизации (node tests/theory.test.js)
```

## Деплой на GitHub Pages

1. Запушить репозиторий на GitHub (ветка `main`).
2. В настройках репозитория: **Settings → Pages → Source → Deploy from a
   branch**, выбрать ветку `main` и папку `/ (root)`.
3. GitHub выдаст адрес вида `https://<username>.github.io/<repo>/`.
   Открывать сайт нужно именно по этому https-адресу (не `file://`) —
   иначе service worker и офлайн-режим не заработают.

## Установка на iPad как приложение (PWA)

1. На iPad открыть Safari (обязательно Safari, не Chrome — только он умеет
   ставить PWA на домашний экран на iOS/iPadOS) и зайти на адрес
   GitHub Pages из шага выше.
2. Один раз зайти на сайт с интернетом — service worker скачает и закэширует
   все файлы. Дальше сайт открывается и работает офлайн.
3. Нажать кнопку **"Поделиться"** (квадрат со стрелкой вверх) в панели Safari.
4. В списке действий выбрать **"На экран «Домой»"** (Add to Home Screen).
5. Подтвердить название и нажать **"Добавить"**.
6. На домашнем экране появится иконка "Guitar Practice" — открывается как
   отдельное приложение, без адресной строки Safari, в полноэкранном режиме.

### Как обновить уже установленное приложение

Service worker работает по стратегии "сначала сеть": если есть интернет, он
всегда берёт свежие файлы с GitHub Pages и обновляет офлайн-кэш; если
интернета нет — отдаёт то, что закэшировано. Поэтому после `git push` обычно
достаточно:

1. Убедиться, что iPad онлайн.
2. **Полностью закрыть приложение** (смахнуть его вверх в диспетчере
   приложений — обычной сменой вкладки внутри приложения обновление не
   подхватится) и открыть иконку заново.

Если после этого всё равно видна старая версия (например, старый service
worker ещё не успел смениться) — повторить перезапуск ещё раз. Если и это
не помогло — удалить иконку с домашнего экрана и добавить заново (шаги 3-5
выше).

## Тесты

Табличная транспонизация аккордов проверяется unit-тестами на 5 известных
тональностях (C-мажор, A-минор, G-мажор, E-минор, F-мажор):

```
node tests/theory.test.js
```
